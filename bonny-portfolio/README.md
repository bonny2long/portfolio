## Setup & Local Development

Clone and run locally:

```bash
# Clone the repo
git clone https://github.com/bonny2long/bonny-portfolio.git

# Move into directory
cd bonny-portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

The project will run at **http://localhost:5173/** by default.

---

## Build for Production

```bash
npm run build
```

The optimized production build will be generated in the `/dist` folder ‚Äî ready for Netlify deployment.
